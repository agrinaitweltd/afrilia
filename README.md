# Afrilia Agro Uganda Limited - Website

Premier agricultural export company delivering quality Ugandan produce to international markets across Europe, Middle East, and beyond.

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```
